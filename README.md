# Netflix-Movie-Dataset-Analysis-Python-EDA-Project
# 🎬 Netflix Movie Data Analysis & Visualization using Python


📌 Overview

This project focuses on analyzing a movie dataset (Netflix-style data) to extract meaningful insights using Python Data Analytics techniques.
It covers data cleaning, preprocessing, feature engineering, and visualization to understand trends in movies.

🎯 Objectives

✔️ Perform data cleaning and preprocessing    
✔️ Analyze movie trends and patterns    
✔️ Identify popular genres and ratings      
✔️ Visualize insights using charts 📊      
✔️ Generate business insights      

🛠️ Tech Stack
🐍 Python      
📊 Pandas      
🔢 NumPy      
📈 Matplotlib      
🎨 Seaborn      

📂 Dataset Features

🎬 Title      
⭐ Popularity

📊 Vote_Average
🎭 Genre      

🔧 Data Preprocessing
Converted Release_Date → Year      
Cleaned data types      
Handled missing values    
Dropped unnecessary columns:

Original_Language    
Poster_Url    

🔄 Feature Engineering
Categorized Vote_Average into:
🔴 Not Popular    
🟡 Below Average    
🔵 Average    
🟢 Popular    

Split and transformed Genre column into individual values

🔍 Key Insights
